Welcome to HX711's documentation!
===========================================

Here you can find a basic documentation for the HX711 library. 

This library is for the hx711 modul to read weight load.
It is build for esp-idf framework and written in C++!

.. note:: 
   At the moment this library is in BETA phase. Future changes on api will be possible.
